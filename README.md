# ⚡ UI Library

A lightweight and flexible React component library built with TypeScript, Vite, and a design system approach.

This project is focused on building reusable UI components with a strong emphasis on:
- Flexibility
- Theming
- Composition
- Developer experience

---

## 🚀 Tech Stack

- React
- TypeScript
- Vite
- CSS Modules
- Bun / npm / yarn / pnpm compatible
